# Theme-based background music

Add MP3 files here to enable background music by theme. ClipVerse picks a theme from your prompt and uses the matching file:

- **calm.mp3** – serene, peaceful, nature, sunset, ocean, etc.
- **cinematic.mp3** – epic, movie, adventure, space
- **nature.mp3** – wildlife, landscape, forest, outdoor
- **upbeat.mp3** – happy, fun, party, cheerful
- **dramatic.mp3** – tense, suspense, action, dark

Use royalty-free music (e.g. from Pixabay Music, Free Music Archive, or your own). If a theme file is missing, no background music is added.
